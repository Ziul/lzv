# Lempel-Ziv
Encode and decode a file using the Lempel-ziv coding algorithm


### Install

```
    $ pip install -e .
```
  
### Using

Encoding a file:

```
    $ lzv <file> [options]
    $ lzv -t <text> [options]
```

Decoding a file:

```
    $ lzv <file.table | file.acod> -d [options]
```
  
More options:

```
    $ lzv --help
```
